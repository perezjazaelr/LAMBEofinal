PHP AJAX

Using PHP with AJAX, we can parse an XML document from a local directory as well as on a server. The following example demonstrates how to parse XML with a web browser.
The client-end script renders a HTML form and defines a JavaScript function for sending a HTTP request to the server with XMLHttpRequest object.
On the server, a PHP script loads the DOM object from the required XML document, fetches the selected course from the $_REQUEST variable, and renders the details of the course chosen as the response back to the client.
