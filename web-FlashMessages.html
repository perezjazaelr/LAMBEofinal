PHP - Flash Messages

Message flashing in a PHP web application refers to the technique that makes certain messages popup on the browser window for the user to receive application’s feedback. To be able to give the user meaningful feedback to his interactions is an important design principle that gives a better user experience.

SYNTAX

1.<?php

2. session_start();

3. if(isset($_SESSION['flash_message'])) {

4. $message = $_SESSION['flash_message'];

5. unset($_SESSION['flash_message']);

6. echo $message;

7. }

8.?>
