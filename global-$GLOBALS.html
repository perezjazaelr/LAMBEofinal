PHP - $GLOBALS

$GLOBALS is one of the "superglobal" or "automatic global" variables in PHP. It is available in all scopes throughout a script. There is no need to do "global $variable;" to access it within functions or methods. $GLOBALS is an associative array of references to all globally defined variables. The names of variables form keys and their contents are the values of an associative array.

SYNTAX

1.<?php

2. $globalVar1 = "Hello";

3. $globalVar2 = "World";

4. function concatenateGlobals() {

5. $combined = $GLOBALS['globalVar1'] . " " . $GLOBALS['globalVar2'];

6. return $combined;

7. }

8. echo concatenateGlobals();

9. ?>
