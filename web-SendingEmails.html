PHP - Sending Emails

PHP has a built-in mail() function to send an email. However, you need to properly configure the "php.ini" settings to be able to do so. First, you must know the SMTP domain of the web hosting platform that you are using. For example, if your website is being hosted on GoDaddy hosting service, the SMTP domain is "smtp.secureserver.net", which you should use in the configuration.

SYNTAX

1.<?php

2. $to = "xyz@somedomain.com";

3. $subject = "This is subject";

4. $message = "<b>This is HTML message.</b>";

5. $message .= "<h1>This is headline.</h1>";

6. $header = "From:abc@somedomain.com \r\n";

7. $header .= "Cc:afgh@somedomain.com \r\n";

8. $header .= "MIME-Version: 1.0\r\n";

9. $header .= "Content-type: text/html\r\n";

10. $retval = mail ($to,$subject,$message,$header);

11. if( $retval == true ) {

12. echo "Message sent successfully...";

13. }else {

14. echo "Message could not be sent...";

15. }

16.?>
