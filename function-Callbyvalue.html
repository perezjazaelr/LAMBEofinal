PHP - Call by value

By default, PHP uses the "call by value" mechanism for passing arguments to a function. When a function is called, the values of actual arguments are copied to the formal arguments of the function’s definition. During the execution of the function body, if there is any change in the value of any of the formal arguments, it is not reflected in the actual arguments.
Actual Arguments − The arguments that are passed in a function call.
Formal Arguments − The arguments that are declared in a function definition.
