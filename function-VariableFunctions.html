PHP - Variable Handling Functions

PHP Variable Handling Functions are the inbuilt PHP library functions that enable us to manipulate and test php variables in various ways.

SYNTAX

1.<?php

2.// Example of isset()

3. $var1 = 10;

4.if (isset($var1)) {

5.echo "\$var1 is set.\n";

6.} else {

7.echo "\$var1 is not set.\n";

8. }

9.// Example of unset()

10.$var2 = 20;

12. echo "\$var2 before unset: $var2\n";

13. unset($var2);

14. // $var2 is now undefined

15.if (isset($var2)) {

16.echo "\$var2 is set.\n";

17.} else {

18.echo "\$var2 is not set.\n";

19.}

20.// Example of empty()

21.$var3 = '';

22.if (empty($var3)) {

23. echo "\$var3 is empty.\n";

24.} else {

25. echo "\$var3 is not empty.\n";

26.}

27.?>
