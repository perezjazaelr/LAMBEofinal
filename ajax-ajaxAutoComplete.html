PHP - AJAX Auto Complete Search

Autocomplete feature is a typeahead mechanism to show input suggestions as the user enters data in the search box provided. It is also called live search because it reacts to the users' input. In this example, we shall use AJAX and XML parser in PHP to demonstrate the use of autocomplete textboxes.
