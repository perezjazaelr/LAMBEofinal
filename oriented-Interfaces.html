PHP - Interfaces

an interface in PHP can be called as a template for classes. We know that when a class is instantiated, the properties and methods defined in a class are available to it. Similarly, an interface in PHP declares the methods along with their arguments and return value. These methods do not have any body, i.e., no functionality is defined in the interface.
A concrete class has to implement the methods in the interface. In other words, when a class implements an interface, it must provide the functionality for all methods in the interface.
An interface is defined in the same way as a class is defined, except that the keyword "interface" is used in place of class.

SYNTAX

1.<?php

2. interface shape {
public function area(): float;
}

3.

4. class square implements shape {
var $side;
public function __construct($arg1) {
$this->side = $arg1;
}
public function area(): float {
return pow($this->side, 2);
}
}

5.

6. class circle implements shape {
var $radius;
public function __construct($arg1) {
$this->radius = $arg1;
}
public function area(): float {
return pow($this->radius,2)*pi();
}
}

7.

8.

9. $sq = new square(5);
echo "Side: " . $sq->side . " Area of Square: ". $sq->area() . PHP_EOL;

10.

11.

12. $cir = new circle(5);
echo "Radius: " . $cir->radius . " Area of Circle: " . $cir->area(). PHP_EOL;

13.

14.?>
