PHP File Handling

PHP - Download File

Most modern browsers allow files of certain types to be downloaded automatically, without any server-side code such as a PHP script. For example, a zip file, or an EXE file. If an HTML hyperlink points to a ZIP or EXE file, the browser downloads it and pops up a save dialog. However, text files, image files, etc., are not downloaded but opened in the browser, which you can save to your local filesystem.
The readfile() Function
To download such files (instead of the browser automatically opening them), we can use the readfile() function in PHP’s built-in function library.

SYNTAX

1.<?php

2.$filePath = 'welcome.png';

3.// Set the Content-Type header to application/octet-stream

4.header('Content-Type: application/octet-stream');

5.// Set the Content-Disposition header to the filename of the downloaded file

6.header('Content-Disposition: attachment; filename="'. basename($filePath).'"');

7.// Read the contents of the file and output it to the browser.

8.readfile($filePath);

9.<?php
