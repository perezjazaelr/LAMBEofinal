PHP - Inheritance

The child class will inherit all the public and protected properties and methods from the parent class. In addition, it can have its own properties and methods.
An inherited class is defined by using the extends keyword.

SYNTAX

1.<?php

2.class Fruit {

3. public $name;

4. public $color;

5. public function __construct($name, $color) {

6. $this->name = $name;

7. $this->color = $color;

8. }

9. protected function intro() {

10. echo "The fruit is {$this->name} and the color is {$this->color}.";

11. }

12.}

13.class Strawberry extends Fruit {

14. public function message() {

15. echo "Am I a fruit or a berry? ";

16. // Call protected method from within derived class - OK

17. $this -> intro();

18. }

19.}

20.$strawberry = new Strawberry("Strawberry", "red"); // OK. __construct() is public

21.$strawberry->message(); // OK. message() is public and it calls intro() (which is protected) from within the derived class

22.<?php
