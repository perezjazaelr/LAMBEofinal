PHP - DOM Parser Example

The DOM extension in PHP comes with extensive functionality with which we can perform various operations on XML and HTML documents. We can dynamically construct a DOM object, load a DOM document from a HTML file or a string with an HTML tag tree. We can also save the DOM document to a XML file, or extract the DOM tree from a XML document.

SYNTAX

1.<?php

2. /*** a new dom object ***/

3. $dom = new domDocument;

4. /*** load the html into the object ***/

5. $dom->loadHTMLFile("hello.html");

6. /*** discard white space ***/

7. $dom->preserveWhiteSpace = false;

8. /*** the table by its tag name ***/

9. $tables = $dom->getElementsByTagName('table');

10. /*** get all rows from the table ***/

11. $rows = $tables[0]->getElementsByTagName('tr');

12. /*** loop over the table rows ***/

13. foreach ($rows as $row) {

14. /*** get each column by tag name ***/

15. $cols = $row->getElementsByTagName('td');

16. /*** echo the values ***/

17. echo 'Designation: '.$cols->item(0)->nodeValue.'<br />';

18. echo 'Manager: '.$cols->item(1)->nodeValue.'<br />';

19. echo 'Team: '.$cols->item(2)->nodeValue;

20. echo '<hr />';

21. }

22.?>
